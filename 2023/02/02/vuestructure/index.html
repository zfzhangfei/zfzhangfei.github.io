1、安装node

Node.js (nodejs.org)

查看版本，有版本号即安装成功

2、安装vue脚手架

npm install -g @vue/cli

3、创建项目(VsCode)

打开终端，进入想要创建项目的文件夹下

注意，最好在管理员模式下创建，否则最后创建可能会报错
